Setting up Octopress under OS X Lion … nothing is simple these days!

My first post.

So I’m expanding my repertoire and came across [http://octopress.org/ Octopress], “A blogging framework for hackers”. It publishes static html (so it’s performant) pages which you â€žcreate using markdown syntax (so it’s easy to code) and can publish straight to github (so your code is open and freely available). Everything is open source and lovely!

The install process seems simple enough, all you need is ruby, git .. but getting your

  1. Get XCode form the App store http://itunes.apple.com/app/xcode/id497799835
  2. Install the XCode CLI Tools https://developer.apple.com/downloads/index.action
  3. Install Homebrew http://mxcl.github.com/homebrew/
  4. Install Git via Homebrew $ sudo brew install git

Now you can look at the [http://octopress.org/docs/setup/ installation documentation] for Octopress, but hold your horses! We’re not running a contemporary linux distro that has all of this stuff already packaged .. this means there’s more prep work to do :(

  1. Time up update your gems! $ sudo gem update –system
  2. You’ll be needing liquid as a dependancy later on, so it’s easier to update that now $ sudo gem install liquid -v ‘2.2.2’
  3. rb-fsevent is another dependancy that needs up be updated before you can proceed … $ sudo gem install rb-fsevent -v ‘0.4.3.1’

  4. sudo xcode-select -switch /Applications/Xcode.app

  5. I’m still running ruby 1.8 $ sudo rvm install ruby-1.9.2-p318
  6. Now switch to the new verison of ruby, it doesn’t happen automaically :-/ $ rvm use 1.9.3
  7. Update your bundles … $ bundle update
  8. now you can install Octopress! $ rake install

Whew :)
